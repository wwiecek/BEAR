# Open Science Collaboration Source Files

Source-package files from the Reproducibility Project: Psychology are kept in
`reproducibility_package/`. The data were obtained from the OSF repository at
<https://osf.io/ytpuq/overview>. Local exploratory outputs are kept in
`derived/`. The paper PDF is kept at the folder root for reference.

The canonical BEAR processor is `process/OSC.R`; it reads
`reproducibility_package/rpp_data.csv` and writes `data/OSC.rds`.
